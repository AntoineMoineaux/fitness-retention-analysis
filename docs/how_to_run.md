# How to Run the Analysis

1. Open the SQLite database in SQLiteStudio
2. Review tables: users, workouts
3. Execute SQL scripts in the following order:
   - 00_schema_overview.sql
   - 01_data_exploration.sql
   - 02_weekly_user_workouts.sql
   - 03_habit_threshold.sql
   - 04_cohort_retention.sql
   - 05_wau_weekly.sql
   - 06_north_star_weekly.sql
   - 07_kpi_summary.sql
4. Load resulting datasets into Power BI
5. Build visuals as shown in the dashboard screenshots

Note: PBIX file is not shared; screenshots and SQL logic are provided instead.
